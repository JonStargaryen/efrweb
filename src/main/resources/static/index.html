Prediction of Early Folding Residues

Protein folding is a little understood process. Interestingly, folding happens stepwise and a defined set of residues initiates the process by folding early. Experimental data on early folding residues is provided by the Start2Fold database. However, for most protein structure no data is available.

This is why we developed this resource which predicts the position of early folding residues from structure data. Specify either a PDB-ID or upload your structure of interest. Several features are computed to describe each residue in the structure and a trained model is used to predict individual residues as early or late folding.

This demonstrates how a rather simple machine learning technique, called generalized matrix learning vector quantization (GMLVQ), can be employed to create a sparse, interpretable classification model. The algorithm is provided as plug-in of the popular Weka machine learning toolbox.
